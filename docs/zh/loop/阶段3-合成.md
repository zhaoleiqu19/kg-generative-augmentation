# 阶段 3 — 合成:现状(stage3-state-of-the-art 中文版)

> 脊柱 stage 3:真正生成规格化的图像(及标注)。最*成熟*的阶段——已有丰富、可直接用于检测的方法菜单。这里的活是挑对工具,而非发明工具。

## 是什么
生成工具箱,四个家族:
1. **Layout/box 条件生成** —— 以 bbox/几何为条件的扩散;标注免费获得。
2. **合成(copy-paste)** —— 生成/抠出孤立目标,过滤,带掩码贴到背景上。
3. **个性化** —— 从少量图像学一个特定稀有主体,在新条件下重新渲染。
4. **通用扩散增强** —— 语义编辑 / 类条件采样(多为分类)。

## 为何重要
这是电梯/电动车图像被造出来的地方。方法选择控制保真度(一个被证明"有用"的杠杆,Azizi 2023),以及**小/遮挡**域差填得多好(→ G3)。

## 现状

### Layout/box 条件(检测)
- ODGEN(Zhu 2024)—— 逐目标条件(视觉块 + 逐目标文本);处理**多类 + 遮挡**;**+25.3% mAP**(domain-specific),COCO +5.6%。最贴合锚点。
- GeoDiffusion(Chen 2023)—— 几何/bbox → prompt;首个证明 L2I 扩散数据有助检测器;训练快 4×。
- ReCon(Zhu 2025)—— 通过感知反馈纠偏 + region 对齐交叉注意力,把 region 可控做到*忠实*(NeurIPS'25 spotlight)。
- AeroGen(Tang 2024)—— 小目标(遥感),水平+旋转框,**内联过滤**,**稀有类 +12.6–17.8%**。

### 合成(copy-paste)
- X-Paste(Zhao 2023)—— SD 生成/爬取目标 + CLIP 过滤 + 贴;长尾(LVIS)上 **+6.8 box AP**。
- 文生图用于检测(Ge 2023)—— 前景+背景+cut-paste,带自动框/掩码。

### 个性化
- DreamBooth(Ruiz 2022)—— 从 3–5 张图学某特定电动车型号,在新光照/姿态/遮挡下重新渲染。

### 通用 / 保真度
- 扩散增强综述(Alimisis 2024)—— 方法+指标分类。· 合成数据改善 ImageNet(Azizi 2023)—— 保真度提升下游准确率。· AI 生成图像作为数据源(Yang 2023)—— 家族 + 47× 成本优势。

## 已解决 vs 开放
- **已解决:** 可直接用于检测的生成已成熟。两个对锚点强而互补的配方:(a) **ODGEN 式逐目标 layout 生成**(遮挡感知、大 mAP),(b) **DreamBooth 个性化目标 → CLIP 过滤的 copy-paste**(稀有类、低风险)。稀有/长尾增益一致(ODGEN、AeroGen、X-Paste)——对稀有的"电梯里电动车"类是鼓舞。
- **开放:** 哪个最能填上**小/遮挡**差距,对我们的区间未测量(→ G3);微小目标的标注紧致度无人涉及;且关键在于,这些全假设*规格/layout 是给定的*——没有一个从诊断推导它(→ G4)。

## 我们的看法
stage 3 不是瓶颈——菜单丰富、数字强,以 ODGEN 和 DreamBooth+X-Paste 为电梯锚点的两个领跑配方。杠杆在上游(诊断 → spec,G2/G4)和测量小目标区间(G3);stage 3 提供那些缺口要驱动的引擎。

# 合成数据有用(还是有害)?(does-synthetic-data-help 中文版)

## 是什么
"有用 vs 有害"之争:何时加入生成/合成图像会改善一个下游视觉模型,何时会让它退化?本节点收集双方证据并带引用的数字,以便后续阶段制定设计规则。

## 为何重要
整个项目押注于生成能改善一个检测器。这个赌注只有在特定条件下才安全;本节点钉住这些条件,使诊断驱动闭环生成*有益的那一类*数据、避开有害的那一类。

## 关键论文

### 有用(HELPS)
- 合成数据改善 ImageNet(Azizi 2023)—— 分类:用扩散样本增强 ImageNet 胜过强 ResNet/ViT 基线;采样分辨率从 256→1024 时准确率 **64.96% → 69.24%**(FID 1.76)。保真度是杠杆。
- 文生图用于检测(Ge 2023)—— **检测/分割**:*仅*用合成数据训练的检测器在 VOC/COCO 上达到"与真实数据相当";合成+真实更好。cut-and-paste 带自动框。
- AI 生成图像作为数据源(Yang 2023)—— 合成数据每张便宜约 47 倍;cut-and-paste 前景合成在其对比中给出最大的逐图检测增益。

### 有用——但只是近持平(怀疑派)
- 大数据迷思(Voetman 2023)—— **检测**:*仅*用微调 SD 合成数据训练的检测器与真实数据基线近持平(苹果园,AP 偏差 **0.09–0.12**)——但是在简单、低遮挡、单类任务上,且用**人工**框。读作"替代品而非放大器":盲合成 ≈ 真实,所以增益需要*针对*。

### 有害 / 局限(HURTS / LIMITS)
- 生成数据放大偏见(Zhang 2024)—— 跨代把生成数据喂回(自我消费闭环)会**放大子群偏见**;即便准确率没掉,公平性也退化。反对盲/迭代增强。

## 综合(暂定规则)
1. **保真度重要** —— 更高质/分辨率样本更有用(Azizi:仅分辨率就 +4.3 个百分点)。
2. **针对性 + 组合 胜过 纯 + 盲** —— 合成*与*真实一起、且瞄准具体缺口,是可靠区间(Ge;参 SafeFix 针对)。
3. **不过滤地迭代有害** —— 反复自我消费放大偏见(Zhang);需要一个过滤步骤(VLM,参 SafeFix)+ 子群监控。
4. **对我们的区间仍开放:** 没有一个"有用"结果隔离了**小/遮挡检测**这一情形——锚点的区间未经验证(喂给全景 G3)。

## 开放问题
- 在什么合成与真实之比下,*检测*(非分类)的有用会转为有害?
- 保真度(Azizi 的杠杆)对检测是否和对分类一样重要?
- 一个 VLM/质量过滤器能否阻止 Zhang 报告的偏见放大,同时保住增益?

## 我们的看法
合成数据在**高保真、与真实数据组合、瞄准真实缺口、且经过过滤**时可靠有用——在**盲、迭代、不过滤**时可靠有害风险。诊断驱动闭环在构造上就站在每一条轴的安全一侧;唯一未经证明的轴是小目标检测本身。
